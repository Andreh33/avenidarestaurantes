# Restaurantes Avenida · «El Paseo»

Web del grupo **Restaurantes Avenida** (Getafe, Madrid). Proyecto de [Latech](https://serviciosonlineweb.com).

- **Fuente de verdad:** [`docs/PROMPT-MAESTRO.md`](docs/PROMPT-MAESTRO.md) — concepto, leyes, plan de bloques B0–B22.
- **Reglas de trabajo para agentes:** [`CLAUDE.md`](CLAUDE.md).
- **Decisiones de estudio:** [`docs/DECISIONES.md`](docs/DECISIONES.md).

## Stack

Next.js 16 (App Router) · React 19 · TypeScript strict · Tailwind CSS v4 · GSAP + Lenis · Motion · Zustand · zod · pnpm. Deploy en Vercel.

## Desarrollo

```bash
pnpm install
pnpm dev
```

> `_ingest/` contiene material original del cliente y credenciales de ingesta: está en `.gitignore` y **jamás se commitea**.
